# Release Notes - 2026-03-24 - v0.1.0-foundation

## Summary
Initial observability foundation release for D0-D5.

## Included
1. Repo scaffold (`module/`, `platform/`, `examples/`, `alerts/`, `dashboards/`, `scripts/`).
2. Module API contract skeleton with required public interfaces and env parsing.
3. Local LGTM + OTEL platform baseline with collector pipeline and health checks.
4. Baseline alerts and dashboards as code.
5. Example Gin + Asynq integration with smoke automation.
6. CI workflows for module, platform, policy, and examples smoke.

## Public API Status
- Added stable API skeleton:
  - `bootstrap.Init`, `bootstrap.Shutdown`
  - `ginx.Middleware`
  - `workerx.AsynqMiddleware`
  - `dbx.WrapGORM`
  - `health.RegisterRoutes`
  - `metrics.RegisterRoute`

## Compatibility
- Backward compatibility preserved (initial baseline release).
- No breaking API changes in this release.

## Operational Notes
- Default local ports are mapped to non-conflicting host ports in `platform/docker-compose.yml`.
- `make smoke` validates required endpoints and baseline telemetry from `examples/`.
